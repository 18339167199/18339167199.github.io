轮播图

使用:
scss:
@import "joweb/src/sass/lib/carousel";

js:
var Carousel = require('lib/carousel');
var carousel = new Carousel({
  el: '.carousel'
  height: 750
  width: 750
})

参数 默认 说明
el (String) 无 元素选择器，以 # 或者 . 开头, 例如 '#app' 选择 id 为 app 的元素，以 . 开头, 例如 '.app', 则是选择document.getElementsByClassName('app')的第一个元素
height (Number / String) 500 轮播图的高度, 传入的数值为 Number 类型则单位为 (px), 也可以采用 String 类型， 例如 '100%', '50%'
width (Number /String) '100%' 同 height
autoplay (Boolean) true 是否自动轮播
interval (Number) 5000 轮播的时间间隔 (ms)
index (Number) 1 开始轮播的位置, 第一个轮播图单元为 1
duration (Number) 1000 轮播动画开始到结束的时间 (ms)
direction (String) 'right' 轮播的方向, 'left' / 'right'
hoverToStop (Boolean) true 鼠标移入是否停止播放
createIndicators (Boolean) true 是否自动生成指示器
indicators (HTMLElement) 无 当 createIndicators 为 false 时, 可以指定一个DOM元素为改轮播图的指示器, 当 createIndicators 为 true 时无效
indicatorsToggle (String) 'click' 指示器触发的方式, 'click': 点击切换轮播图, 'hover': hover切换轮播图
