tooltips

使用:
scss:
@import "joweb/src/scss/tooltips";

js:
var Tooltips = require('lib/tooltips');
var tooltips = new Tooltips()

根据 HTML 标签上的属性初始化
<div class="jw-tooltips" data-content="内容" data-title="标题" data-toggle="激活的方式, click / hover" data-placement="弹出的位置, left / right / top / bottom">
</div>

参数 默认 说明
el (String) 无 不传入改参数时, 自动获取 所有 class 为 'jw-tooltips' 的元素初始化
元素选择器，以 # 或者 . 开头, 例如 '#app' 选择 id 为 app 的元素，以 . 开头, 例如 '.app', 则是选择document.getElementsByClassName('app')的第一个元素
